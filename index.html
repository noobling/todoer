Automatically assigns appropriate team members to todos

The algorithm works by looking for the user's skills in the todo item's heading, description and tags. The more matches found the more suitable the user is for that todo item. For example a user enters Front end and Back end for their skill set. The program will search for Front end and Back end in the todo item matching every word that is similar to Front end and Back end. The user with the most skill matches in the todo item is deemed to most appropriate.

Displays the most relevant todos to you first

Having a shared todo list can be a nightmare to manage because the number of todos becomes overwhelming. We have solved this problem similar to how Facebook presents your news feed. We place the most relevant todos first for you. So you only deal with whats most important to you

The algorithm for this works like so: First find all todo items where which you are assigned to then order todos by priority and days left before it is due. Priority takes precedence over days left. Priority is color coded as red for high, orange for medium and green for low.

Simple yet powerful filtering to enhance your work flow

As we have previously mentioned shared todo lists become very large so having a powerful filtering feature is a must. However there is often a trade off with usuability, but we have engineered our filtering mechanism so there is no such compromise.

You can search through the entire todo list instantly with the search bar. It matches words in the filter with words present in the todo item.

You can click the most relevant filter option to find all items most relevant to you. How this is done has been explained in the section above. We have deliberately abstracted away this because there should be no need for the user to figure out what they want. An intelligent program should be able to find out themselves.

Click on highest priorty to find todo items that are most critical for the entire project even if you are not assigned to it. This is so that you can get an overview of what your team is doing so you can help them out with their todos.
